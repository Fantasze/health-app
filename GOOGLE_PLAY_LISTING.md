## 上架流程

### 第 1 步：创建 Google Play 开发者账号
```
1. 访问：https://play.google.com/console
2. 登录 Google 账号
3. 支付 $25 注册费
4. 填写开发者资料
5. 等待审核（1-3 天）
```

### 第 2 步：创建应用
```
1. 点击"创建应用"
2. 填写应用名称
3. 选择默认语言（简体中文）
4. 确认应用类型（应用）
5. 确认是否免费（免费）
```

### 第 3 步：填写应用信息
```
1. 商店列表
   - 上传应用图标
   - 上传截图
   - 填写描述
   - 选择分类

2. 内容分级
   - 填写问卷
   - 获得分级

3. 目标受众
   - 选择年龄组
   - 选择国家

4. 新闻应用声明
   - 选择"否"

5. 数据安全
   - 填写数据收集声明
```

### 第 4 步：上传 APK
```
1. 进入"发布" → "生产"
2. 创建新版本
3. 上传 APK 文件
4. 填写版本信息
5. 保存
```

### 第 5 步：提交审核
```
1. 检查所有信息
2. 点击"提交审核"
3. 等待审核（1-7 天）
4. 审核通过后自动上架
```
