## 🎯 修复任务列表

### 任务 1：加固 Firestore 规则
**优先级：** P0 - 立即
**预计时间：** 30 分钟
**测试：** 验证权限规则生效

**步骤：**
1. 读取当前 firestore.rules
2. 编写安全的规则（基于用户认证）
3. 部署到 Firebase
4. 测试规则是否生效

**验收标准：**
- [ ] 未认证用户无法写入数据
- [ ] 用户只能读写自己的数据
- [ ] 只有管理员能写销售码
- [ ] 教练信息可公开读

### 任务 2：添加 Firebase Authentication
**优先级：** P0 - 立即
**预计时间：** 1 小时
**测试：** 登录/注册流程正常

**步骤：**
1. 启用 Firebase Auth（邮箱/电话）
2. 修改注册页面添加 Auth
3. 修改登录页面添加 Auth
4. 添加认证状态检查
5. 更新所有需要认证的页面

**验收标准：**
- [ ] 用户可以注册
- [ ] 用户可以登录
- [ ] 未登录用户无法访问受限页面
- [ ] 用户 ID 与 Firestore 数据关联

### 任务 3：修复销售码兑换逻辑
**优先级：** P0 - 立即
**预计时间：** 30 分钟
**测试：** 无法重复兑换

**步骤：**
1. 添加事务保护（runTransaction）
2. 检查销售码是否已使用
3. 检查用户是否已兑换
4. 原子更新销售码和用户数据
5. 添加错误处理

**验收标准：**
- [ ] 销售码只能使用一次
- [ ] 用户只能兑换一次
- [ ] 并发兑换不会重复
- [ ] 错误提示友好
